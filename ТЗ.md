# Тестовое задание Playable Developer (2D)

## Цель

Разработать клон следующего playable креатива:

**Референс:** https://playbox.play.plbx.ai/playoff/runner

## Ресурсы

- **Важно: логика должна быть разработана с нуля на выбранном вами игровом JS/TS based движке или фреймворке (Cocos Creator, Pixi.js и т.п.).**
- **Важно**: качество результата должно быть **не хуже** исходника
- **Ассеты:** Извлеките ассеты напрямую из веб-страницы референса
- **Недостающие ассеты:** Используйте Nano Banana или другие генеративные AI-инструменты для создания недостающих ассетов
- **AI-инструменты:** Вы можете использовать любых AI-агентов или ассистентов для помощи в разработке

## Требования

Ваша работа должна соответствовать следующим критериям:

1. Исходный код должен быть сохранен
2. **Билд Единый HTML-файл**
    - Вся игра должна быть упакована в один HTML-файл
    - Все ассеты, скрипты и стили должны быть встроены
3. **Качество:**
    1. Игра должна открываться на полный экран для возможности тестирования адаптивной верстки.
    2. Не должно быть фризов (используйте energy save мод на телефоне или low tier mobile в Chrome Dev Tools для тестирования)
    3. Должна быть реализована финишная лента с эффектом веревки.
4. **Размер бандла**
    - Финальный размер бандла не должен превышать **5 МБ**
5. **Система контроля версий**
    - Используйте Git-репозиторий для отслеживания прогресса
    - Требуется несколько осмысленных коммитов (работы с одним коммитом не принимаются)
    - Сообщения коммитов должны четко описывать внесенные изменения
6. **Публикация**
    - используй github.io или vercel.com чтобы опубликовать билд

## Задача со звёздочкой: Рескин ⭐

Замените **главного персонажа** и **фон** на альтернативные ассеты, соответствующие другому сеттингу **на ваш выбор**.

Можно сгенерировать анимацию в NanoBanana в Google AI Studio.

**Примеры:**

- Вариант с Марго Робби
- Вариант с Toly
